# Refugee-Assistant-Switzerland
AI Form Helper + Simple Administrative Q&amp;A

## Goal
Help newly arrived refugees in Switzerland understand official form questions and get simple first-step guidance.

## Tech Stack
- Python
- Streamlit
- OpenAI API
- SQLite

## Setup
1. Clone the repo
2. Create a virtual environment
3. Install requirements
4. Add `.env`
5. Run `streamlit run app.py`
