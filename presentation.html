How It Works

Lightweight RAG Pipeline

RAG = Retrieval-Augmented Generation. The app finds real Swiss sources first, then asks the AI to answer using them — so every response is grounded in official information.

  • User asks a question in any language
  • Backend scores 27 Swiss official sources
  • Top 3 most relevant sources are selected
  • Sources + history sent to AI model
  • AI responds in the user's own language

Example: "Can I work with Permit F?" in Arabic → finds SEM source → responds in Arabic.

Logic Flow

How the App Makes Decisions

  • Every message checked for API key first
  • Question analyzed for permit type (N/F/B/C/S)
  • If permit found → strict source search
  • If no permit → general keyword search
  • Sources found → injected into AI prompt
  • No sources → AI uses its own knowledge
  • API error → user gets helpful message
  • Every answer saved to database
