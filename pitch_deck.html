The Solution

Refugee Assistant Switzerland

A free, 24/7 AI chatbot — in the user's own language.

  • Multilingual: Responds in Arabic, French, German, Tigrinya and more — automatically
  • Comprehensive: Covers permits N/F/B/C/S, asylum process, work rights, integration, healthcare, education, family reunification & appeals
  • Trusted sources: Grounded in 27 official Swiss sources (SEM, ch.ch, FIDE, OSAR)
  • Always available: No appointment needed — works 24/7 in the browser
  • Free: No app download, no login, no cost for the user

How It Works

Lightweight RAG Pipeline

The app retrieves relevant official sources and injects them into the LLM prompt — ensuring accurate, grounded answers.

🧑 User InputAny language
→
🔍 Keyword Search27 Swiss sources in SQLite
→
🧠 Groq LLMLLaMA 3.3-70b + system prompt
→
💬 ResponseIn user's language

Example: User asks "Can I work with Permit F?" in Arabic → System finds SEM Permit F source → LLM answers in Arabic with accurate work rules.
